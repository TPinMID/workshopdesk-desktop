# WorkshopDesk (Desktop)

WorkshopDesk — настольное приложение для учёта работ в мастерской: заявки/заказы, диагностика, запчасти и отчёты.
Цель — чтобы у мастера всё было в одном окне: что делали, что намерили, что поставили и сколько это стоило.

## Как выглядит
Типичный экран: слева список заказов, в центре карточка заказа, справа быстрые действия (закрыть, печать, экспорт).
Скриншот (заготовка): `assets/screenshots/main.png`

## Ключевые возможности
- Карточка заказа: клиент, объект ремонта (авто/устройство), симптомы, выполненные работы
- Диагностика: замеры (напряжение/ток/сопротивление), заметки, прикреплённые фото
- Запчасти и расходники: списание в заказ, остатки, себестоимость
- Отчёты: за период, по мастерам, по типам работ, экспорт в CSV/PDF

## Технологии (план)
- Python + PySide6 (GUI)
- SQLite (локальная база)
- Экспорт: CSV/PDF

## Быстрый запуск (когда появится код)
1) `python -m venv .venv`
2) `pip install -r requirements.txt`
3) `python src/main.py`
