# Feature: Diagnostics (Диагностика)

Модуль “Диагностика” хранит измерения и наблюдения по заказу так, чтобы их можно было:
1) быстро внести во время работы,
2) легко сравнить “до/после”,
3) показать клиенту в отчёте.

## Что входит
- Таблица замеров (параметр → значение → единицы → комментарий → дата/время)
- Чек-лист диагностики (галочки “проверено”)
- Вложения: фото/скрин осциллограммы/документ

## Пример замеров
- Напряжение АКБ (В)
- Ток потребления (А)
- Падение напряжения на массе (В)
- Сопротивление обмотки (Ом)

## Готовность (Definition of Done)
- Замер добавляется/редактируется/удаляется
- Замеры привязаны к заказу
- Есть экспорт замеров в отчёт
