# Feature: Reports (Отчёты и экспорт)

Модуль “Отчёты” превращает хаос работ в понятные цифры.
Основная цель — быстро ответить на вопросы:
“сколько заработали?”, “что делали чаще?”, “кто на чём сидит по времени?”.

## Варианты отчётов
- Выручка/прибыль за период
- По мастерам (кол-во заказов, средний чек)
- По типам работ
- Топ запчастей/расходников по списаниям

## Экспорт
- CSV (для Excel/Google Sheets)
- PDF (для клиента / для печати)

## Готовность (Definition of Done)
- Отчёт строится за выбранный диапазон дат
- Есть кнопка “Экспорт CSV”
- PDF формируется по шаблону “Акт/Отчёт по заказу”
